Belajar HTML dan CSS

HTML dibuat untuk membuat tampilan website CSS digunakan untuk mengatur tampilan

Materi HTML

  • h1 untuk judul
  • p untu paragraf
  • img untuk gambar
  • tabel untuk tabel
  • input untuk form
  • li untuk list/daftar
Materi CSS
  1. Margin,Pading,Position untuk mengatur jarak antara elemen HTML
  2. Display untuk mengatur elemen HTML agar tampil atau tidak tampil,mendatar,atau menurun
  3. Style untuk mengatur font
